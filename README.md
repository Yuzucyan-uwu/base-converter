# base-converter  (C++ GUI)

A simple base converter with GUI using C++.

## Features
- Convert decimal to base 2~35
- GUI with input box and button
- Supports A-Z for digits > 9

## Tech
- C++
- EasyX graphics library

## How to Run
1. Install EasyX
2. Compile with your C++ compiler
3. Run

## Example
Input: 255, base 16  
Output: FF
